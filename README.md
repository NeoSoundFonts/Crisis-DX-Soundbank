# Crisis-DX-Soundbank
Libre recreation of Crisis GeneralMIDI 3.01, hosted by SP

It was started since Yingchun Soul is doing Aspirin DX Soundbank.

Original Crisis GeneralMIDI is free for home usage only, **can't be used for commercial purposes nor be redistributed without buying license (which is impossible at nowadays), and cannot be modified**, this version is libre and free-for-commercial use and modify.
